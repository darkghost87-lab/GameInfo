## 📖 درباره‌ی پروژه

**GameVerse** یک وبسایت کامل برای معرفی و دانلود بازی‌های ویدیویی است که با هدف ارائه‌ی تجربه‌ای حرفه‌ای و جذاب برای گیمرها طراحی شده است. این پروژه شامل صفحات متعدد، اسلایدرهای تعاملی، هدر پویا و طراحی کاملاً ریسپانسیو می‌باشد.

> ⚠️ **توجه:** این پروژه صرفاً در سمت کاربر (فرانت‌اند) توسعه‌یافته و هنوز به بک‌اند متصل نیست.

## 🎯 نحوه‌ی استفاده
ناوبری در وبسایت:

1.منوی اصلی: در هدر وبسایت، بین صفحات مختلف جابه‌جا شوید
2.منوی موبایل: در اندازه‌های کوچک، منو به صورت همبرگری نمایش داده می‌شود (header.js)
3.اسلایدرها: با کلیک روی دکمه‌های قبلی/بعدی یا کشیدن، اسلایدها را تغییر دهید

مشاهده‌ی صفحات:
صفحه‌ی اصلی (index.html): شامل هیرو سکشن با افکت ذرات و اسلایدر بازی‌ها

درباره‌ی ما (about.html): اطلاعات تیم و هدف وبسایت

قوانین (Rules.html): قوانین و مقررات استفاده

ورود (login.html): فرم ورود و ثبت ‌نام کاربران

## 🔧 جزئیات فنی
هدر تعاملی (header.js):
منوی دسکتاپ با افکت‌های هاور

منوی همبرگری برای موبایل

انیمیشن باز و بسته شدن نرم

افکت ذرات (particles.js):
پس‌زمینه‌ی پویا با ذرات متحرک

تایپوگرافی مناسب برای تمام دستگاه‌ها
